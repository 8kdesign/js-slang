slang
=====
An open-source interpreter for *The Source*.

Usage
-----
The REPL is still under development.

To install,

.. code-block::

  $ git clone https://github.com/ningyuansg/slang.git
  $ cd slang
  $ npm install
  $ npm run build
  $ npm install -g

To invoke the repl, run the command `slang` anywhere:

.. code-block::

  $ slang
  >
