\section*{Changes}

Compared to Source \S 2, Source \S 3 has the following changes:
\begin{itemize}
\item variable declaration
\item variable assignment
\item array/object assignment
\item \textbf{\texttt{while}} loops
\item \textbf{\texttt{for}} loops
\item \textbf{\texttt{break}} and \textbf{\texttt{continue}}
\item array/object access expressions
\item literal object expressions
\item dot abbreviation (see Section \textbf{Objects})
\end{itemize}
